м»ҙнҸ¬л„ҢнҠё кё°л°ҳ CSS мһ‘м„ұм—җ м Ғн•©н•ң Styled-Component #

  • кё°мЎҙ CSS л¬ёлІ•мңјлЎңлҸ„ мҠӨнғҖмқј мҶҚм„ұмқҙ м¶”к°Җлҗң React м»ҙнҸ¬л„ҢнҠёлҘј л§Ңл“Ө мҲҳ мһҲмқҢ

  • Automatic critical CSS

    • Styled Component лҠ” нҷ”л©ҙм—җ м–ҙл–Ө м»ҙнҸ¬л„ҢнҠёк°Җ л ҢлҚ”л§Ғ лҗҳм—ҲлҠ”м§Җ м¶”м Ғн•ҙм„ң н•ҙлӢ№н•ҳлҠ” м»ҙнҸ¬л„ҢнҠём—җ лҢҖн•ң мҠӨнғҖмқјмқ„ мһҗлҸҷмңјлЎң мӮҪмһ…н•©лӢҲлӢӨ.
    • мҪ”л“ңлҘј м Ғм ҲнһҲ л¶„л°°н•ҙ лҶ“мңјл©ҙ мӮ¬мҡ©мһҗк°Җ м–ҙн”ҢлҰ¬мјҖмқҙм…ҳмқ„ мӮ¬мҡ©н•  л•Ң мөңмҶҢн•ңмқҳ мҪ”л“ңл§ҢмңјлЎң нҷ”л©ҙмқҙ лқ„мӣҢм§ҖлҸ„лЎқ н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

  • No class name bugs

    • Styled Component лҠ” мҠӨмҠӨлЎң мң лӢҲнҒ¬н•ң className мқ„ мғқм„ұн•©лӢҲлӢӨ.
    • className мқҳ мӨ‘ліөмқҙлӮҳ мҳӨнғҖлЎң мқён•ң лІ„к·ёлҘј мӨ„м—¬мӨҚлӢҲлӢӨ.

  • Easier deletion of CSS

    • кё°мЎҙм—җлҠ” лҚ” мқҙмғҒ мӮ¬мҡ©н•ҳм§Җ м•Ҡкұ°лӮҳ мӮӯм ңн•ң м»ҙнҸ¬л„ҢнҠём—җ н•ҙлӢ№н•ҳлҠ” мҠӨнғҖмқј мҶҚм„ұмқ„ м ңкұ°н•ҳкё°мң„н•ҙ CSS нҢҢмқј м•Ҳмқҳ classNameмқ„ мқҙлҰ¬м ҖлҰ¬ м°ҫм•„м•ј н–ҲмҠөлӢҲлӢӨ.
    • н•ҳм§Җл§Ң Styled Component лҠ” лӘЁл“  мҠӨнғҖмқј мҶҚм„ұмқҙ нҠ№м • м»ҙнҸ¬л„ҢнҠёмҷҖ м—°кІ°лҗҳм–ҙ мһҲкё° л•Ңл¬ём—җ л§Ңм•Ҫ м»ҙнҸ¬л„ҢнҠёлҘј лҚ” мқҙмғҒ мӮ¬мҡ©н•ҳм§Җ м•Ҡм•„ мӮӯм ңн•  кІҪмҡ° мқҙм—җ лҢҖн•ң мҠӨнғҖмқј мҶҚм„ұлҸ„ н•Ёк»ҳ мӮӯм ңлҗ©лӢҲлӢӨ.

  • Simple dynamic styling

    • classNameмқ„ мқјмқјмқҙ мҲҳлҸҷмңјлЎң кҙҖлҰ¬н•  н•„мҡ” м—Ҷмқҙ React мқҳ props лӮҳ м „м—ӯ мҶҚм„ұмқ„ кё°л°ҳмңјлЎң м»ҙнҸ¬л„ҢнҠём—җ мҠӨнғҖмқј мҶҚм„ұмқ„ л¶Җм—¬н•ҳкё° л•Ңл¬ём—җ к°„лӢЁн•ҳкі  м§ҒкҙҖм Ғмһ…лӢҲлӢӨ.

  • Painless maintenance

    • м»ҙнҸ¬л„ҢнҠём—җ мҠӨнғҖмқјмқ„ мғҒмҶҚн•ҳлҠ” мҶҚм„ұмқ„ м°ҫм•„ лӢӨлҘё CSS нҢҢмқјл“Өмқ„ кІҖмғүн•ҳм§Җ м•Ҡм•„лҸ„ лҗҳкё° л•Ңл¬ём—җ мҪ”л“ңмқҳ нҒ¬кё°к°Җ м»Өм§ҖлҚ”лқјлҸ„ мң м§ҖліҙмҲҳк°Җ м–ҙл өм§Җ м•ҠмҠөлӢҲлӢӨ.

  • Automatic vendor prefixing

    • к°ңлі„ м»ҙнҸ¬л„ҢнҠёл§ҲлӢӨ кё°мЎҙмқҳ CSS лҘј мқҙмҡ©н•ҳм—¬ мҠӨнғҖмқј мҶҚм„ұмқ„ м •мқҳн•ҳл©ҙ лҗ  лҝҗмһ…лӢҲлӢӨ.
    • мқҙмҷёмқҳ кІғл“ӨмқҖ Styled Component к°Җ м•Ңм•„м„ң мІҳлҰ¬н•ҙ мӨҚлӢҲлӢӨ.

DOM referenceлҘј мһҳ нҷңмҡ©н•  мҲҳ мһҲлҠ” useRef #

// example(1)
import React, { useRef } from "react";

const Focus = () => {
  const firstRef = useRef(null);
  const secondRef = useRef(null);
  const thirdRef = useRef(null);

  const handleInput = (event) => {
    console.log(event.key, event);
    if (event.key === "Enter") {
      if (event.target === firstRef.current) {
        secondRef.current.focus();
        event.target.value = "";
      } else if (event.target === secondRef.current) {
        thirdRef.current.focus();
        event.target.value = "";
      } else if (event.target === thirdRef.current) {
        firstRef.current.focus();
        event.target.value = "";
      } else {
        return;
      }
    }
  };

  return (
    <div>
      <h1>нғҖмһҗм—°мҠө</h1>
      <h3>к°Ғ лӢЁм–ҙлҘј л°”лҘҙкІҢ мһ…л Ҙн•ҳкі  м—”н„°лҘј лҲ„лҘҙм„ёмҡ”.</h3>
      <div>
        <label>hello </label>
        <input ref={firstRef} onKeyUp={handleInput} />
      </div>
      <div>
        <label>world </label>
        <input ref={secondRef} onKeyUp={handleInput} />
      </div>
      <div>
        <label>codestates </label>
        <input ref={thirdRef} onKeyUp={handleInput} />
      </div>
    </div>
  );
};

export default Focus;

Today’s takeaway #

  • мҳӨлҠҳмқҖ Reactмқҳ styleмқ„ л§Ңл“ңлҠ” лІ•мқ„ л°°мӣ лӢӨ.

  • м—¬лҹ¬ л°©лІ•л“Өмқҙ мғқкІЁлӮ¬кі  м•һмңјлЎң мғқкІЁлӮ  кІғмқҙлӢӨ.

  • к°Ғ л°©лІ•л“Өмқҳ мһҘлӢЁм җмқ„ м•Ңкі  м Ғм ҲнһҲ нҷңмҡ©н•ҳмһҗ!

    л°©лІ• мһҘм җ лӢЁм җ
    CSS - мқјкҙҖлҗң нҢЁн„ҙмқ„ к°–кё° м–ҙл ӨмӣҖ, !importantмқҳ лӮЁмҡ©
    SASS(preprocessor) ліҖмҲҳ/н•ЁмҲҳ/мғҒмҶҚмқ„ нҷңмҡ©н•ҙ мһ¬мӮ¬мҡ©(CSS кө¬мЎ°нҷ”) л””лІ„к№… м–ҙл өкі , м»ҙнҢҢмқјн•ң CSSнҢҢмқјмқҙ кұ°лҢҖн•ҙм§җ
    BEM м „мІҳлҰ¬ кіјм • л¶Ҳн•„мҡ” м„ нғқмһҗ мқҙлҰ„мқҙ мһҘнҷ©н•ҳкі , нҒҙлһҳмҠӨ лӘ©лЎқмқҙ л„Ҳл¬ҙ л§Һм•„м§җ
    Styled-Component(CSS-in-JS) CSSлҘј м»ҙнҸ¬л„ҢнҠё м•ҲмңјлЎң мәЎмҠҗнҷ”, л„Өмқҙл°ҚмқҙлӮҳ мөңм Ғнҷ”лҘј мӢ кІҪм“ё н•„мҡ”X л№ лҘё нҺҳмқҙм§Җ лЎңл“ңм—җ л¶ҲлҰ¬н•Ё

  • м•„м§ҒмқҖ к°җмқҙ мһҳ мһЎнһҲм§Җ м•ҠмңјлӮҳ мқҙнӢҖ лҸҷм•Ҳ мӢӨмҠөмқ„ нҶөн•ҙ мқөнһҗ кІғмқҙлӢӨ.
