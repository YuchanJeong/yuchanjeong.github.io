CSS мӨ‘кёү #

  • CSS Selector
    • selector1 + selector2лҠ” selector1мқҳ нҳ•м ң мҡ”мҶҢ мӨ‘ лӢӨмқҢ selector2
    • selector1 ~ selector2лҠ” selector1мқҳ нҳ•м ң мҡ”мҶҢ мӨ‘ лӘЁл“  selector2
    • selector:last-childлҠ” мһҗмӢқ мҡ”мҶҢмқё selector мӨ‘ л§Ҳм§Җл§ү мҡ”мҶҢ
    • selector1 > selector2:lastChildлҠ” selector1мқҳ мһҗмӢқ мҡ”мҶҢмқё selector2 мӨ‘ л§Ҳм§Җл§ү мҡ”мҶҢ
    • selector:last-child(n)мқҖ мһҗмӢқ мҡ”мҶҢмқё selector мӨ‘ л§Ҳм§Җл§үм—җм„ң nлІҲм§ёмҡ”мҶҢ
    • selector1 > selector2:nth-last-child(2n+1)мқҖ selector1мқҳ мһҗмӢқ мҡ”мҶҢмқё selector2 мӨ‘ л§Ҳм§Җл§үм—җм„ң нҷҖмҲҳ лІҲм§ё мҡ”мҶҢ
    • selector:first-of-typeмқҖ мһҗмӢқмҡ”мҶҢмқё selector мӨ‘ мІ« л“ұмһҘ мҡ”мҶҢ
    • selector:last-of-typeмқҖ мһҗмӢқмҡ”мҶҢмқё selector мӨ‘ л§Ҳм§Җл§ү л“ұмһҘ мҡ”мҶҢ
    • selector:nth-of-type(n)мқҖ nлІҲм§ёмһҗмӢқ мҡ”мҶҢмқё selector(selector мӨ‘м—җм„ң nлІҲм§ё)
    • selector1:not(selector2)лҠ” selector2к°Җ м•„лӢҢ selector1
  • кё°ліё мҠӨнғҖмқјл§Ғмқ„ м ңкұ°н•ҳлҠ” CSS мҪ”л“ң
    • body { margin: 0; padding: 0; }
  • Flex box

  • мЎ°кёҲ лҚ” л°°мӣҢм„ң м •ліҙлҘј л¶Ҳлҹ¬мҳӨлҠ” л°©мӢқмңјлЎң л§Ңл“Өкі  мӢ¶лӢӨ.
  • vwмҷҖ vhлҘј нҷңмҡ©н•ҙм„ң л°ҳмқ‘нҳ•мңјлЎң л§Ңл“Өм—ҲлӢӨ.

Today’s takeaway #

  • мҳӨлҠҳмқҖ CSSлҘј нҷңмҡ©н•ҙм„ң м§Ғм ‘ мӣ№ нҷ”л©ҙмқ„ кө¬м„ұн•ҙ ліҙм•ҳлӢӨ.
  • мқҙлЎ м ҒмңјлЎңлҠ” мһҳ м•Ңкі  мһҲм—Ҳм§Җл§Ң, м§Ғм ‘ нҷ”л©ҙмқ„ л§Ңл“Өм–ҙліҙлӢҲ м–ҙл Өмӣ лӢӨ. лӮҙк°Җ мӣҗн•ҳлҠ”лҚ°лЎң мһҗм—°мҠӨлҹҪкІҢ кө¬м„ұн•ҳкі  мһ‘лҸҷ мӢңнӮӨлҠ”лҚ° м–ҙл ӨмӣҖмқҙ мһҲм—ҲлӢӨ.
  • нҠ№нһҲ HTML кө¬мЎ°лҘј л§Ңл“Ө л•Ң, м–ҙл–»кІҢ нҡЁмңЁм ҒмңјлЎң л¬¶м–ҙм„ң л§Ңл“Өм§Җ кі лҜјмқҙ л§Һмқҙ лҗҳм—ҲлӢӨ.
  • CSSм—җм„ңлҸ„ м„ нғқмһҗлҘј м–ҙл–»кІҢ нҷңмҡ©н•ҳлҠ” кІғмқҙ нҡЁмңЁм Ғмқјм§Җ м•„м§ҒмқҖ мһҳ лӘЁлҘҙкІ лӢӨ.
  • мқҙлІҲ мЈјл§җм—җлҠ” CSSлЎң м—¬лҹ¬ л Ҳмқҙм•„мӣғл“Өмқ„ л§Ңл“Өм–ҙліҙл©ҙм„ң м—°мҠөн•ҙм•јкІ лӢӨ.
