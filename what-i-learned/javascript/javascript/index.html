3. Etc #

1) Destructuring Assignment(кө¬мЎ° л¶„н•ҙ н• лӢ№)

л°°м—ҙ/к°қмІҙмқҳ мҶҚм„ұмқ„ н•ҙмІҙ нӣ„ к·ё к°’мқ„ к°ңлі„ ліҖмҲҳм—җ н• лӢ№н•ҳлҠ” н‘ңнҳ„мӢқ

Array

const arr = [1, 2, 3, 4, 5];
const [a, , b, ...rest] = arr;
// aлҠ” 1, bлҠ” 3, restлҠ” [4, 5]

Object

const obj = { name: "X", age: 10, gender: "male" };
const { name, gender: sex, address = "Korea" } = obj;
// nameлҠ” "X", sexлҠ” "male", addressлҠ” "Korea"

2) Spread Syntax(м „к°ң кө¬л¬ё)

…dataмқҳ нҳ•нғңлЎң л¬ёмһҗм—ҙВ·л°°м—ҙВ·к°қмІҙлҘј нҺјм№ҳлҠ” н‘ңнҳ„мӢқ

String

const str = "Che";
// [...str]лҠ” ["C", "h", "e"]
// {...str}лҠ” { "0": "C", "1": "h", "2": "e" }

Array

const arr1 = [1, 2, 3];
// [...arr1]мқҖ [1, 2, 3], !== arr1
// {...arr1}лҠ” { "0": 1, "1": 2, "2": 3 }
const arr2 = [4, 5, 6];
// [...arr1, ...arr2]лҠ” [1, 2, 3, 4, 5, 6]

Object

const obj = { a: 1, b: 2, c: 3 };
// {...obj1}мқҖ { "a": 1, "b": 2, "c": 3 }, !== obj1
const obj2 = { c: 1, d: 2, e: 3 };
// {...obj1, ...obj2}лҠ” { "a": 1, "b": 2, "c": 1, "d": 2, "e": 3}

Rest Prams

const func = (team, ...members) => ({ нҢҖ: team, л©ӨлІ„: members });
const myTeam = func("KT", "Smeb", "Score", "Ucal", "Deft", "Mata");
/* 
myTeamмқҖ {
  нҢҖ: "KT",
  л©ӨлІ„: ["Smeb", "Score", "Ucal", "Deft", "Mata"],
};
*/
