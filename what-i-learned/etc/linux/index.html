2. 경로 탐색 #

Command Details
pwd 현재 절대 경로 보기
ls
ㅤ[경로명]
ㅤ-l
ㅤ-a		 현재 경로 목록 보기
- 경로 지정
- 자세히 보기
- 숨겨진 목록 보기
which [프로그램명] 해당 프로그램의 실행 경로 보기
cd
ㅤ[경로명]
ㅤ..
ㅤ~
ㅤ-
해당 경로로 이동
상위 경로로 이동
최상위 경로로 이동
직전 경로로 이동
find . -type directory|file -name “[검색어]” 현재 및 하위 경로의 특정 폴더|파일 찾기
(Ex. “*A*”, “*.md”)
권한 (ls -l로 볼 수 있음) 사용 권한 (Ex. drwxr-xr-x)
  • d(directory) | -(non-directory)
  • user group other
  • r(read) | w(write) | x(execute)
권한 변경: chmod
  1. Symbolic method (Ex. chmod a=r-x project)
    • u(user) | g(group) | o(other) | a(all)
    • +(add) | -(remove) | =(set exact)
    • r(read) | w(write) | x(execute)
  2. Absolute form (Ex. chmod 740 index.js)
    • user group other
    • 4(r)2(w)1(x)
